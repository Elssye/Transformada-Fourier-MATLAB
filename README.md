# Simulación y Análisis de Señales con Transformada de Fourier en MATLAB

## Autor

Elssye Elizabeth Canizales Martinez

## Descripción

Este proyecto utiliza MATLAB para simular señales en el dominio del tiempo y analizar su comportamiento en el dominio de la frecuencia usando la Transformada de Fourier.

## Señales analizadas

* Señal senoidal
* Pulso rectangular
* Función escalón

## Herramienta utilizada

* MATLAB
* Función fft()

## Resultados

Se obtuvieron gráficas en el dominio del tiempo y frecuencia que permiten observar las componentes frecuenciales de cada señal.

## Conclusión

La Transformada de Fourier permite analizar señales e identificar las frecuencias que las componen, siendo una herramienta fundamental en ingeniería y procesamiento de señales.

## Cómo ejecutar

Abrir MATLAB y ejecutar el archivo:

fourier.m
